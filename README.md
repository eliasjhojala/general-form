# general-form

Rails engine for standardized forms with one field definition source (`*Fields`), reused by both view rendering and controller strong parameters.

Default direction: use `general-form` wherever it fits, extend it with backward-compatible improvements when needed, and use fully custom forms only for truly custom workflows.

**Full documentation:** [docs/GUIDE.md](docs/GUIDE.md) — setup, field types, helpers, controller handling, and conventions used in host applications.
